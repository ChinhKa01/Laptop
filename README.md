# Đề tài: Xây dựng website bán hàng cho công ty công nghệ EZone Shop

## Tính năng

- Khách hàng chưa có tài khoản:
    + Xem tất cả sản phẩm
    + Xem chi tiết sản phẩm
    + Đăng kí tài khoản
- Khách hàng đã có tài khoản:
    + Đăng nhập
    + Đăng xuất
    + Lấy lại mật khẩu
    + Xem sản phẩm
    + Xem chi tiết sản phẩm
    + Tìm kiếm sản phẩm
    + Xem thông tin cá nhân
    + Thay đổi thông tin cá nhân
    + Đổi mật khẩu
    + Thêm sản phẩm vào giỏ hàng
    + Xóa sản phẩm trong giỏ hàng
    + Đặt hàng
    + Hủy đặt hàng
    + Xem lịch sử đặt hàng
    + Xem chi tiết đơn hàng
- Quản lý:
    + Đăng nhập
    + Đăng xuất
    + Xem thông tin cá nhân
    + Thay đổi thông tin cá nhân
    + Đổi mật khẩu
    + Thêm sửa xóa sản phẩm
    + Thêm sửa xóa nhà sản xuất
    + Thêm sửa xóa danh mục sản phẩm
    + Duyệt và hủy đơn đặt hàng

## Công nghệ sử dụng
- HTML, CSS, JS, PHP
- MySQL

## Cài đặt
Tải project và tạo database `15_php_2022` sau đó import file `15_php_2022.sql` trong thư mục docs của dự án.
